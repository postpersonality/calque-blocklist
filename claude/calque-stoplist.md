# Стоп-лист калек — самые частые

Не употреблять в русской прозе; замена по контексту, для sense-калек запрет только в кальковом значении. Виды калек, правила и полный список — раздел «Russian Communication Quality» в ~/.claude/CLAUDE.md. Файл порождается скриптом, руками не править.

- «гейт» → контроль качества, проверка
- «калька» (латиницей в прозе) → заимствование
- «скоуп» → охват, рамки задачи, объём
- «барьер» (в кальковом значении) → контроль качества, проверка
- «маппить» → сопоставлять, отображать
- «партиал» → дочерний шаблон, частичный шаблон
- «коллизия» (в кальковом значении) → конфликт, пересечение
- «view» (латиницей в прозе) → представление, вид, тематический документ
- «дёргать» (в кальковом значении) → вызывать
- «pending» (латиницей в прозе) → отложенные, ожидающие, очередь ожидающих
- «матчиться» → совпадать
- «инъектор» → внедритель, вставщик, хук-внедритель
- «layout» (латиницей в прозе) → раскладка, макет
