# Черновой список калек — полный сырой улов

Сырой высокополнотный улов кандидатов в кальки из собственной прозы агента по всем сессиям всех проектов Claude Code. Это ЧЕРНОВИК: намеренно широкий, с шумом (имена, родные слова, освоенные заимствования). Валидация и рекомендации — в `calque-analysis.md`.

## Как собрано

- Корпус: `~/.claude/projects/**/*.jsonl`, 2170 файлов, ~445 МБ; 21 575 текстовых сообщений ассистента, из них 10 380 преимущественно русскоязычных.
- Из прозы вырезаны блоки кода, инлайн-бэктики, мета-обсуждения калек (чтобы цитаты не раздували частоты).
- Три источника сигнала: (1) широкий сканер прозы — частотные словари кириллических и латинских токенов; (2) штатный `mine_calques.py` — окна вокруг маркеров коррекций (для pseudo-term/calque-sense, не детектируемых по форме); (3) `find_calque_uses.py` — контекст по неоднозначным терминам.
- Кириллические кандидаты отобраны по паттернам транслитерации (`дж`, `оу`, `-инг`, `-бэк`, `-абельн`, `эй*`, `-шн$`) и курируемому списку англо-корней + морфологии выдуманных глаголов; словоформы схлопнуты в леммы.

Всего различимых лемм-групп: 325. Уже покрыто списком: 36.
