## 7. Из штатного майнера коррекций (pseudo-term / calque-sense)

Маркерный `mine_calques.py` подтвердил уже покрытые (скоуп→охват, тайтл→заголовок, маппить→сопоставлять, иммутабелен→неизменяем, экспонирует→публикует, faithful copy→точная копия) и дал один новый кандидат:

## Как вносить

Для каждой калеки из раздела 1 после финального решения пользователя:

```bash
python3 ~/.claude/skills/calque-blocklist/scripts/merge_entry.py --json '{
  "<лемма>": { "replace": ["<замена>"], "kind": "<translit|coined-verb>", "note": "<источник>", "detect": ["<основа>"], "tier": "<substitute|hint>" }
}'
```

Перед выбором `detect` прогнать основу через `find_calque_uses.py`, чтобы не зацепить родные слова из раздела 3 (например, основа `спан` зацепит «экспансия», `клон` — «склоняться»).
